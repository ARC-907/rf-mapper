# Model weights

This directory is intentionally empty in source. The MiDaS depth-inference model
(`model_small.onnx`, ~64 MB) is downloaded by the install script.

To fetch manually:

- See `scripts/setup.sh` (Linux / macOS) or `scripts/setup.ps1` (Windows) for
  the download command.
- The model file goes here as `weights/model_small.onnx`.

Upstream source: [Intel ISL MiDaS](https://github.com/isl-org/MiDaS).
