# Quick Help – SparkMind RF Desktop

### Getting Started
- **Open Image** – Ctrl+O: Load a DEM or raster file.
- **Analyze** – Ctrl+R: Run RF simulation on loaded terrain.
- **Export Loss** – Ctrl+E: Save 2D path loss result.
- **F1**: Toggle this help panel.

### Tips
- Hover over any button to learn what it does.
- You must load a DEM before Analyze or Export buttons will activate.
- 3D View, Vectors, and Path Profile require completed analysis.
